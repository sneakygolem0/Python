# Python Decorator Վարժություններ

## 1. Ժամանակի Չափման Դեկորատոր

Գրել դեկորատոր, որը կչափի և կտպի ֆունկցիայի կատարման տևողությունը վայրկյաններով։

---

## 2. Սկիզբ և Ավարտ Հաղորդագրությունների Դեկորատոր

Գրել դեկորատոր, որը ֆունկցիայի կանչից առաջ և հետո կտպի հաղորդագրություն։

**Օրինակ:**
```
Function started 
... ֆունկցիայի աշխատանքը ... 
Function finished
```

---

## 3. Արգումենտների և Արդյունքի Տպման Դեկորատոր

Գրել դեկորատոր, որը ֆունկցիան կանչելիս կտպի՝

- Ֆունկցիայի անունը
- Փոխանցված բոլոր արգումենտները
- Ֆունկցիայի վերադարձրած արդյունքը

---

## 4. Կանչերի Հաշվիչ Դեկորատոր

Գրել դեկորատոր, որը կհաշվի, թե քանի անգամ է ֆունկցիան կանչվել։

---

## 5. Rate Limiter Դեկորատոր

Գրել դեկորատոր, որը թույլ կտա ֆունկցիան կանչել ոչ ավել, քան վայրկյանը 1 անգամ։

Եթե ավելի հաճախ են կանչում, պետք է տպի զգուշացում։

---

## 6. Cache (Memoization) Դեկորատոր

Գրել դեկորատոր, որը կպահի (cache) ֆունկցիայի հաշվարկած արդյունքները։

Եթե նույն արգումենտներով ֆունկցիան կրկին կանչվի, այն պետք է վերադարձնի պահպանված արժեքը՝ առանց ֆունկցիայի մարմինը նորից աշխատեցնելու։
