# Ռեկուրսիվ Ֆունկցիաներ - Վարժություններ

## 1. Թվի Թվանշանների Գումար (Sum of Digits)

Գրել ռեկուրսիվ ֆունկցիա `sum_digits(n)`, որը կհաշվի տրված ամբողջ դրական թվի բոլոր թվանշանների գումարը։

- **Օրինակ:** `sum_digits(458)` պետք է վերադարձնի 4 + 5 + 8 = 17

- **Ռեկուրսիվ Տրամաբանություն:**
  - Վերջին թվանշանը ստանալու համար օգտագործել `n % 10`։
  - Մնացած մասը ստանալու համար օգտագործել `n // 10`

- **Բազային Դեպք:** Եթե n-ը փոքր է 10-ից (մեկանիշ է), վերադարձնել n-ը։

## 2. Զույգ Թվերի Ֆիլտրում (Recursive Even Filter)

Գրել ռեկուրսիվ ֆունկցիա `filter_even(lst)`, որը ստանում է ամբողջ թվերի լիստ և վերադարձնում **նոր լիստ**, որտեղ կան միայն **զույգ թվերը**։ Արգելվում է օգտագործել ցիկլեր (`for`, `while`) կամ `filter()` ֆունկցիան։

- **Օրինակ:** `filter_even([1, 4, 3, 6, 9, 2])` պետք է վերադարձնի `[4, 6, 2]`։

- **Ռեկուրսիվ Տրամաբանություն:**
  1. Ստուգել առաջին էլեմենտը։
  2. Առաջին էլեմենտը ավելացնել մնացած մասի զույգ թվերով լիստին (եթե այն զույգ է)(լիստը կարող է հայտարարված լինել ինչպես գլոբալ, այնպես էլ կարող է փոխանցվել ֆունկցիային որպես default արգումենտ)։

- **Բազային Դեպք:** Եթե լիստը դատարկ է, վերադարձնել դատարկ լիստ `[]`։

## 3. Տողում Որոշակի Սիմվոլի Քանակը (Count Char)

Գրել ռեկուրսիվ ֆունկցիա `count_char(s, char)`, որը կհաշվի, թե տրված տողում քանի անգամ է հանդիպում տրված սիմվոլը։ Չի թույլատրվում օգտագործել տողերի ներկառուցված մեթոդները, ինչպես `count()`, կամ ցիկլեր։

- **Օրինակ:** `count_char("հայկական", "ա")` պետք է վերադարձնի `3`։

- **Ռեկուրսիվ Տրամաբանություն:**
  1. Ստուգել, թե արդյոք տողի առաջին սիմվոլը հավասար է փնտրվող սիմվոլին։
  2. Եթե հավասար է, ապա արդյունքը 1 + մնացած մասի քանակը։ (քանակը հայտարարված է գլոբալ, օգտագործել global keyword-ը փոփոխականի արժեքը փոխելու համար)
  3. Եթե հավասար չէ, ապա արդյունքը 0 + մնացած մասի քանակը։

- **Բազային Դեպք:** Եթե տողը դատարկ է, վերադարձնել 0

## 4. Պալինդրոմ Ստուգում (Is Palindrome)

Գրել ռեկուրսիվ ֆունկցիա `is_palindrome(s)`, որը կստուգի, թե արդյոք տրված տողը պալինդրոմ է (առաջից և հետևից նույնն է կարդացվում)։ Ֆունկցիան պետք է վերադարձնի True կամ False։ Արգելվում է օգտագործել տողի շրջման որևէ մեթոդ։

- **Օրինակներ:**
  1. `is_palindrome("aba")` - True
  2. `is_palindrome("racecar")` - True
  3. `is_palindrome("hello")` - False

- **Ռեկուրսիվ Տրամաբանություն:** Պալինդրոմ է, եթե՝
  1. Տողի **առաջին** և **վերջին** սիմվոլները հավասար են։
  2. Եվ տողի **մնացած մասը** (առանց առաջին և վերջին սիմվոլների) նույնպես պալինդրոմ է։

- **Բազային Դեպք:** Եթե տողը դատարկ է կամ ունի ընդամենը **մեկ սիմվոլ**, այն պալինդրոմ է (վերադարձնել True):

## 5. Աստիճանի Հաշվարկ (Recursive Power)

Գրել ռեկուրսիվ ֆունկցիա `power(base, exponent)`, որը կհաշվի բազային թվի աստիճանը base^(exponent): Օգտագործել միայն բազմապատկում։ Արգելվում է օգտագործել Python-ի `**` օպերատորը կամ `pow()` ֆունկցիան։ Ենթադրել, որ ցուցիչը դրական ամբողջ թիվ է։

- **Օրինակ:** `power(2, 4)` պետք է վերադարձնի 2 * 2 * 2 * 2 = 16։

- **Ռեկուրսիվ Տրամաբանություն:** base^(exponent) = base * base^(exponent-1)։

- **Բազային Դեպք:** Եթե ցուցիչը 0 է, ապա արդյունքը 1 է (base^0 = 1)։
