# Python Վարժություններ - Ֆիլտրում և Zip

## 1. Թվերի Ֆիլտրում

Տրված է թվերից կազմված լիստ։
```python
numbers = [5, 12, 18, 7, 25, 4, 30, 9]
```

1. Ֆիլտրել և տպել միայն 10-ից մեծ թվերը։
2. Ֆիլտրել և տպել միայն 3-ի բազմապատիկ թվերը։
3. Ֆիլտրել և տպել եռանիշ թվերը
4. Ֆիլտրել և տպել միայն զույգ, երկնիշ թվերը
5. Ֆիլտրել և տպել միայն պարզ թվերը
6. Ֆիլտրել և տպել միայն, պարզ եռանիշ թվերը

---

## 2. Տողերի Ֆիլտրում

Տրված է տողերից կազմված լիստ։
```python
names = ["James", "table", "Helloooo!!!", "Rob", "How are you??"]
```

1. Ֆիլտրել և տպել միայն այն տողերը, որոնք սկսվում են 'A' տառով։
2. Ֆիլտրել և տպել միայն այն տողերը, որոնք պարունակում են ! կամ ? սիմվոլները։
3. Ֆիլտրել և տպել միայն այն տողերը, որոնք ունեն զույգ երկարություն։
4. Ֆիլտրել և տպել միայն այն տողերը որոնց երկարությունը 5-ից մեծ է։
5. Ֆիլտրել և տպել այն տողերը, որոնց վերջում դրված է անիմաստ բացատ

---

## 3. Zip - Համընկնումների Հաշվում

Տրված են երկու list՝ actual_results և expected_results։ Օգտագործելով zip, գտիր քանի տեղում սկզբնական և սպասվող արդյունքները համընկնում են։
```python
actual = [1, 2, 3, 4, 5]
expected = [1, 1, 3, 0, 5]
```

**Արտածել** → 3 (համընկնում ունեցող index-ները)

---

## 4. Zip + Filter - Ուսանողների Ֆիլտրում

Տրված է երկու լիստ, առաջինը ուսանողների անուններն են, երկրորդը նրանց ստացած գնահատականները
```python
students = ["Anna", "Mark", "Diana", "John"]
scores = [48, 95, 72, 30]
```

Օգտագործելով zip + filter, վերադարձնել միայն այն ուսանողների անունները, որոնց score ≥ 60։

---

## 5. Zip - Հաջորդական Տարբերություններ

Տրված է իրական թվերից կազմված լիստ։ Տպել հաջորդական տարրերի տարբերությունը օգտագործելով zip ֆունկցիան։

---

## 6. Zip - Զույգ Ինդեքսներ և Տողեր

Տրված է երկու list։ Տպել միայն այն զույգերը, որտեղ՝
- index-ը զույգ է
- և զանգվածի համապատասխան ինդեքսի տակ գտնվող էլեմենտները տողեր են
```python
a = ["hi", 1, "hello", "test"]
b = ["world", "x", "python", 5]
```

**Արդյունք** → միայն `("hello", "python")`

---

## 7. Zip - Տողերի Զույգեր

Տրված են երկու լիստ, որտեղ տարբեր տիպերի տարրեր կան։
```python
x = ["hi", 12, "ok", 3.5, "python", 1]
y = ["wow", 5, "yes", 8, "code", "x"]
```

Վերադարձնել միայն այն զույգերը, որտեղ
- երկու էլեմենտներն էլ string են
- և երկարությունը >= 3 է։

---

## 8. Zip + Filter - IT Բաժնի Աշխատակիցներ

Տրված է երկու լիստ — անուններ և բաժիններ
```python
names = ["Anna", "Mike", "John", "Sophia", "Artur"]
departments = ["IT", "HR", "IT", "Finance", "HR"]
```

Օգտագործելով zip + filter, տպել միայն այն մարդկանց անունները, որոնք աշխատում են "IT"-ում։
